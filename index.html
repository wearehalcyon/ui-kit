Halcyon UI Kit - This is a powerful tool for developing web application interfaces. This library is based on native CSS and jQuery functions. Full support for modern browsers and top-notch performance. Halcyon UI Kit is inspired by the latest Apple products, so you may well notice the similarities of the library elements with the latest versions of MacOS, iPadOS or iOS. This library is distributed free of charge, you can already install it in your project and start using it. Thank you for choosing Halcyon UI Kit!

Start Guide

Styles

Include Font Awesome icons <link rel="stylesheet" href="./src/css/all.min.css">
Include main style file <link rel="stylesheet" href="./src/css/halcyon.ui.kit.css">

Scripts

Include jQuery (1.12.4) library if not included <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
Include main scripts file <script src="src/js/jquery.halcyon.ui.kit.js"></script>

You can now get started. Use navigation in the left sidebar.

Grid System

Our grid system is somewhat similar to Bootstrap, but slightly different. Below you can familiarize yourself with the basic principle of work.

Container

Container: max-width: 1400px;
Use: <div class="hn-container"></div>

Row

Use: <div class="hn-container"> <div class="hn-row"></div> </div>
